# 电商系统总览 — ecommerce-tech-dark

Reference multi-page site demonstrating the `/ai-draw --mode site` feature (v0.2).

Generated by [ai-draw](https://github.com/stone-yu/ai-draw-skill) on 2026-05-13.

## What this example demonstrates

- **Multi-page drill-down**: A root `index.html` with drillable components (↗) linking to depth-1 service pages; one of those pages (`user-service.html`) has a further drillable component linking to a depth-2 page (`auth-module.html`).
- **Breadcrumb navigation**: Every subpage has an `<nav class="ai-draw-breadcrumb">` above the diagram so users can navigate back up the hierarchy.
- **Cross-page theme sync**: All pages share `data-site-id="ecommerce-tech-dark"`. Pressing `T` on any page cycles themes and persists the choice to `localStorage`; reopening any other page in the site picks up the same theme automatically.
- **Asset path depth rules**: Each page uses a relative path to `assets/` that is correct for its directory depth (3 `../` levels for index, 4 for depth-1, 5 for depth-2).

## How to test it

```bash
open site/examples/ecommerce-tech-dark/index.html
```

1. Click the **↗** indicator on **UserService**, **OrderService**, or **PaymentService** to drill into their internal architecture pages.
2. Inside **User Service**, click **↗** on **AuthModule** to reach the deepest page (depth 2).
3. Use the breadcrumb at the top of each subpage to navigate back.
4. Press **T** on any page to cycle through the 3 recommended themes (`tech-dark`, `blueprint`, `cyberpunk-neon`). Open another page in a new tab — it will load with the same theme you selected.

## Keyboard shortcuts

| Key | Action |
|---|---|
| `T` | Cycle 3 recommended themes (tech-dark → blueprint → cyberpunk-neon) |
| `Shift+T` | Cycle all 8 themes |
| `F` | Fullscreen |
| Click `⋯` | Export current page to PNG |

## Known limitations

- Theme sync uses `localStorage` and is **per-browser**. Cross-device sync is not supported.
- The `/ai-draw fill <slug>` command (for regenerating failed subpages) is not yet implemented in v0.2.

## Re-theme entire site

```bash
/ai-draw redo --style blueprint
```

## Add a subpage

```bash
/ai-draw add --to ecommerce-tech-dark --under order-service "Inventory reservation sub-system"
```
